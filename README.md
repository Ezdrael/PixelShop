# PixelShop
PixelShop — це демонстраційний проєкт веб-додатку для електронної комерції, розроблений для імітації функціональності онлайн-магазину. Він демонструє використання PHP для бекенду з архітектурою MVC (Модель-Представлення-Контролер) та сучасних фронтенд-технологій для створення динамічного та зручного для користувача інтерфейсу.
